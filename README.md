 Chatbot with Sentiment Analysis

This project implements a conversational chatbot that performs **conversation-level** and **message-level sentiment analysis** using Python.

The assignment includes:
- **Tier 1 (Mandatory):** Conversation-level sentiment analysis  
- **Tier 2 (Optional):** Per-message sentiment analysis + trend detection
